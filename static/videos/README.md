# Videos Folder

## Place your video background here

**Required file:** `fitness-bg.mp4`

### Download Instructions:

1. Go to: https://www.pexels.com/video/3125907/
2. Click "Free Download"
3. Choose "HD (1280x720)" or "Full HD (1920x1080)"
4. Save as: `fitness-bg.mp4`
5. Place in this folder

### File Structure:
```
videos/
└── fitness-bg.mp4  ← Your video here
```

### Specifications:
- Format: MP4 (H.264)
- Resolution: 1280x720 or 1920x1080
- File size: Under 10MB recommended
- Duration: 10-30 seconds (loops automatically)

### Test:
After adding the video, visit:
http://localhost:5000/buyer/home

The video should play in the hero section background!
